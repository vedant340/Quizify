# QuizPro

Online quiz application built using basic full stack technologies.

## Features
- Multiple choice questions
- Quiz submission
- Score display

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Run Project
npm install
npm start

Open in browser:
http://localhost:3000
